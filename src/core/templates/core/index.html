Django

El framework web para perfeccionistas con plazos de entrega.

Django es un framework web de alto nivel, escrito en Python, que fomenta el desarrollo rápido y un diseño limpio y pragmático.

Fue construido por desarrolladores experimentados para encargarse de gran parte de las complicaciones del desarrollo web, permitiéndote concentrarte en escribir tu aplicación sin necesidad de reinventar la rueda.

Ridículamente rápido

Fue diseñado para ayudar a los desarrolladores a llevar sus aplicaciones desde el concepto hasta la finalización lo más rápido posible.

Seguro

Ayuda a evitar errores comunes de seguridad como inyección SQL, XSS y CSRF de forma automática.

Escalable

Es capaz de manejar incluso los picos de tráfico más intensos (utilizado por Instagram, Pinterest y Disqus).
